# 📚 Guia de Publicação - GitHub Pages

## ⚡ Passo a Passo para Colocar o Site Online

Siga estes passos simples para publicar seu site no GitHub Pages.

---

## 🔧 PASSO 1: Preparar o Repositório no GitHub

### 1.1 Criar um novo repositório
1. Acesse [github.com](https://github.com)
2. Clique no ícone **"+"** (canto superior direito)
3. Selecione **"New repository"**

### 1.2 Preencher os dados
- **Repository name**: `meu-direito-previdenciario`
- **Description**: `Site informativo sobre direitos previdenciários`
- **Public**: ✅ Marque como público
- **Initialize this repository**: ❌ Deixe desmarcado

### 1.3 Criar o repositório
Clique em **"Create repository"**

---

## 📤 PASSO 2: Fazer Upload dos Arquivos

### 2.1 Você está na página do repositório vazio
Procure pelo botão **"uploading an existing file"** (em azul)

### 2.2 Selecionar os arquivos
Clique e selecione TODOS os arquivos e pastas:

```
✅ index.html
✅ README.md
✅ GUIA_PUBLICACAO.md
✅ styles/
✅ scripts/
✅ images/
```

### 2.3 Fazer o upload
- Arraste os arquivos ou clique para selecionar
- Clique em **"Upload"**
- Aguarde o upload terminar

### 2.4 Fazer commit
- Em **"Commit message"** escreva: `Initial commit: Meu Direito Previdenciário`
- Clique em **"Commit changes"**

---

## ⚙️ PASSO 3: Ativar GitHub Pages

### 3.1 Acessar Settings
1. Na página do seu repositório, clique em **"Settings"** (engrenagem)

### 3.2 Procurar por Pages
No menu da esquerda, clique em **"Pages"**

### 3.3 Configurar Source
1. Em **"Source"**, selecione **"Deploy from a branch"**
2. Em **"Branch"**, selecione **"main"**
3. Na pasta, selecione **"/ (root)"**
4. Clique em **"Save"**

---

## ⏳ PASSO 4: Aguardar o Deploy

1. GitHub Pages vai processar os arquivos
2. Aguarde 5-10 minutos
3. Você verá a mensagem: **"Your site is live at..."**

---

## ✨ PASSO 5: Seu Site Está Online!

### 5.1 Acessar o site
A URL será:
```
https://seu-usuario.github.io/meu-direito-previdenciario/
```

### 5.2 Compartilhar
Copie a URL e compartilhe com qualquer pessoa!

---

## 🔄 PASSO 6: Atualizar o Site

Se você fizer mudanças nos arquivos:

### Opção A: Pelo Site do GitHub
1. Abra o arquivo no repositório
2. Clique no ícone de **lápis** (Edit)
3. Faça as mudanças
4. Clique em **"Commit changes"**

### Opção B: Fazer Upload de Novo
1. Clique em **"Add file"** > **"Upload files"**
2. Selecione o arquivo atualizado
3. Clique em **"Commit changes"**

## 🆘 Problemas?

### "Não vejo a opção Pages"
- Verifique se o repositório é **Public**
- Se for Private, mude em Settings > Danger Zone

### "Site não está funcionando"
- Aguarde 10 minutos
- Pressione **F5** para atualizar
- Limpe o cache do navegador

### "Vejo apenas o README"
- Isso é normal! GitHub Pages mostra o README por padrão
- Seu site em `index.html` está funcionando normalmente
- Acesse: `https://seu-usuario.github.io/meu-direito-previdenciario/`
